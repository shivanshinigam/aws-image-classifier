## üåê Live Demo Flow

1.  Upload an image (JPEG/PNG)
2.  Inference via SageMaker (pretrained model)
3.  Top-3 predictions with % confidence
4.  Image stored in S3
5.  Result saved to DynamoDB
6.  Email sent via SNS with prediction
